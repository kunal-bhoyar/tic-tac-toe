# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Two player game (Player X and Player O)
- Winner detection
- Draw prevention
- Reset game button
- Interactive UI
- Simple and clean design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 📂 Project Structure

tic-tac-toe/
│
├── index.html     # Game structure
├── style.css      # Styling
├── app.js         # Game logic
└── README.md      # Project documentation

## 🎮 How to Play

1. Open `index.html` in your browser.
2. Player O starts the game.
3. Click any empty box to place your symbol.
4. Match 3 symbols in a row to win.
5. Press **Reset Game** to play again.
